Principal Rules of DalsanTV

Welcome to DalsanTV. The rules of DalsanTV are made for making the community comfortable. The rules are not made for make wars, and the mission is to moderate the entire website. Everyone needs to follow the rules of DalsanTV including normal users, secondary moderators, and primary moderators. If someone violates those rules, they will have the consequence that the admin or founder takes a punishment for the guilty.

Punishment methods applied for the guilty

If you recieve a ban, You are free to log out, log in, see content of every user and create a new account if this time you will promise to respect the actual rules of DalsanTV. If it was contradicted your promise and you back to violate the principal rules, people can report you as: "Ban Evasor".

Classic: Temporal Ban

Temporal bans makes you are unaviable to interact, see notifications, view other content of other users if you're logged in and even add, edit or delete multimedia content. This type of temporal bans are adjusted from 1 day to 12 months.

Classic: Permanent Ban

Is the same thing that you are unaviable to do that previous things, but you will not have the possibility of recover your account. So in this case, you're banned forever. If you think that we make a mistake on your ban and you didn't violate the main Rules, you can appeal directly to the admin, so it will check if its worth to unban your account or not.

Critical: IP Ban

This only happens in very critical behaviors on an account. IP is assigned so the website will detect the unauthorized ip and the user who has connected to the same IP, will have totally no access to the website forever. Only option: appeal that IP ban if we really make a mistake on the IP ban.

2. Blocking Rules

In other sites you can block an user by any reason, but in DalsanTV, blocking an user is controlled by the admin and the moderators; so in this site, blocking an user will not stop to interact with this user. The admin will controll your block in three types: Total, Partial and Unable; this three types of blocking will apply for the targeted user. You cannot block the admin by any reason and this is the same case with the moderators. If any user blocks some mentioned user, you can block a 100%, be a partial block or unable to block this mentioned user. Anybody can not see if this user was targeted for know which type of block has now.

Total Block. Total block is the most common of blocking an user and this type of block is applied for all new users by default. The blocker will just block the user, it confirms if block this user. If it accepts the block, both blocker and the blocked user will not interact, view or follow mutualy until the blocker changes his/her opinion.

Partial Block. Partial block is applied by the users who act normal and had followed the rules of DalsanTV; partial blocks will only be targeted by the Admin and the moderators. If the blocker blocks the blocked user, are only unable to interact and follow mutually, but they can still be able to watch his/her content, profile and description; not to see his/her followers and to favourite it.

Unable Block. The blocker can not block an user by any reason, even if it's fair by him/her opinion. The admin can only control if target the user as unable to block, but moderators cannot do this thing. Admins, Moderators and super-users can not be blocked by any reason. If the Blocker wants to block an user, it will just appear a warning that says: "You can not block this user", knewing that this user can not be blocked.
